# MiniSwap Frontend

## 功能
- 连接 MetaMask 钱包
- 添加流动性
- 移除流动性
- Token 兑换

## 部署信息
- 网络：Polkadot Paseo Testnet
- TokenA: 0x3bEEbBe939bEB221bdC7A8baA81fEa69295043A8
- TokenB: 0x298FA4226C8880fAccACB844dc4bc83483969D21
- MiniSwap: 0x4544F33f362C55E15F13fbc92312b635c8693Db6

## 本地运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问：http://localhost:3000

## 使用步骤

1. 点击 "Connect MetaMask" 连接钱包
2. 切换到 Polkadot Paseo 测试网
3. 使用界面进行添加流动性、兑换、移除流动性操作
